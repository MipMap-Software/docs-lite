## Settings

![](/img/en-img/settings.png)

### Disk space

>![](/img/en-img/diskspace.png)
Displays the remaining disk space for the current working directory. When available space is low, clean up storage or change the working directory promptly to avoid project rebuild failures caused by insufficient disk space.

### License

>![](/img/en-img/license3.png)
When the license validity is less than fifteen days, the remaining days will be displayed. Please monitor the remaining days to avoid project rebuild failures due to license expiration.


### Settings

Click ![](/img/en-img/setting.png) to open the user settings panel.

#### Preferences

- Working Directory: Click the edit icon to choose the project storage path; check "Move existing projects" to move all current project files to the specified path.

- Basemap Service: Switch the basemap; an internet connection is required to load it.

- Length Unit: Switch the length unit; measurement labels will update accordingly.

- Area Unit: Switch the area unit; measurement labels will update accordingly.

- Volume Unit: Switch the volume unit; measurement labels will update accordingly.


- Auto Merge Cameras: Options are Allow / Ask / Deny; when importing photos, automatically merge photos from the same camera. "Ask" will only auto-merge if the device serial number, photo resolution, and focal length all match.

- User Experience Improvement Program: When enabled, your device information along with necessary diagnostics and logs will be sent to MipMap to help improve user experience and product services.


![](/img/en-img/settings2.png)


#### License management

- Replace License: Replace the license purchased for the current account.

- Unbind: Unbind the license from the current device; after unbinding, the activation code can be bound to another device.

>Note: If you need to switch devices, please unbind the activation code in this panel first!

- Update: Refresh the current license status information.

- Enter Activation Code: Enter a purchased activation code and bind it to the current device.

![](/img/en-img/License4.png)



#### Plugin management

Gaussian Splatting Plugin: After installation, used to generate Gaussian splatter models; can be uninstalled.

Machine Learning Plugin: After installation, used to improve model accuracy and optimize model quality; can be uninstalled.


![](/img/en-img/plugins.png)


#### About

Displays the current Software Version and SDK Version; if a new version is available, click to update.


![](/img/en-img/about.png)
