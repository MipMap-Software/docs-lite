After reconstruction, you can annotate and measure points on 3D results on the map. Gaussian splatter results cannot be measured.

> **Note:** If photos do not include POS position information, you can only view the results; measurement is not available.

## Point measure

![](/img/en-img/point.png)

### Steps

- Click the Point tool, then left-click on the result to select the point to measure. Press `Esc` to exit measurement mode.
- The left panel shows the point coordinates: longitude, latitude, elevation, and orthometric height (requires selecting the vertical datum).
- Click "View Camera" to list all cameras that captured this point (requires the Dataset → Photos layer to be enabled).
- You can edit the point label name and color.
