## View results

Click "View Results" to open the `result` output folder for the task.

![](/img/cn-img/result.png)

### 3D outputs

The `3D` folder contains all 3D output products.

![](/img/cn-img/3D文件夹.png)

- `model-b3dm`: 3D Tiles (B3DM) model output.
- `model-fbx`: FBX model output.
- `model-glb`: GLB model output.
- `model-gs-ply`: Gaussian model in PLY format.
- `model-gs-sog`: Gaussian model in SOG format.
- `model-gs-sog-tile`: Gaussian model in SOG tile format.
- `model-obj`: OBJ model output.
- `model-osgb`: OSGB model output.
- `model-ply`: PLY model output.
- `model-skp`: SKP model output.
- `point-las`: LAS point cloud output.
- `point-pnts`: PNTS point cloud output.

### Photogrammetry outputs

The `AT` folder stores photogrammetry results. `mvs` contains the original SfM results; `mvs_undistort` contains undistorted SfM results.

![](/img/cn-img/企业微信截图_20260604123031.png)

### Log files

The `logs` folder contains project log files.

> If reconstruction fails, send the log files to technical support to help diagnose the error.

![](/img/cn-img/企业微信截图_20260604123249.png)

### Thumbnails

The `thumbnail` folder stores task thumbnails.

![](/img/cn-img/缩略图.png)

- `camera_n_residual`: camera residual distribution image.
- `rgb_thumbnail`: stores the standard-resolution thumbnail image.
- `overlap_map`: overlap/coverage distribution image.
