## Display controls

![](/img/en-img/displaycontrols.png)

![](/img/en-img/3D.png): Browse 3D model and 3D point cloud results.

![](/img/en-img/GS.png): Browse Gaussian splatting results.

![](/img/en-img/mapsnapshot.png): Quickly capture and save the current view as a snapshot.

![](/img/en-img/resetmap.png): Reset to the default view.

![](/img/en-img/zoom.png): Zoom in/out the map.

![](/img/en-img/thumbnail.png): Capture the current view as the task thumbnail.

![](/img/en-img/view.png): Save the current camera view as the default; clicking Reset will return to this view.

**When viewing 3D results:**

![](/img/en-img/top.png): Show top-down (plan) view of the model.

![](/img/en-img/side.png): Show side view of the model.



![](/img/en-img/crop.png): Click Crop to draw a polygon on the map and display only the 3D results within that area. Left-click to add vertices; double-click to finish.

![](/img/en-img/restore.png): Click Restore to end cropped view.

**When viewing GS results:**

![](/img/en-img/skybox.png): Toggle skybox display.

![](/img/en-img/showhelp.png): Show Gaussian model browsing shortcuts; use keys to control navigation of Gaussian models.
