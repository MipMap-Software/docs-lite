## Project Related

![](/img/en-img/projectrelated.png)

### Project Search

![](/img/en-img/search.png)Enter the full project name or keywords to search for projects.

### Project Filter

![](/img/en-img/filter.png)Click the filter dropdown to select display filter criteria.


- TAG: Display projects containing the selected tags.
- Deselect All: Clear all selected filter conditions.



![](/img/en-img/filter2.png)


### Project Sorting

![](/img/en-img/sorting.png)Click the sort dropdown to select sorting rules.

- Name: All projects on the main interface will be sorted alphabetically from A to Z.
- Creation Time: All projects will be sorted from newest to oldest by creation time.
- Update Time: All projects will be sorted from newest to oldest by last modified time.

![](/img/en-img/sorting2.png)

### Project Display Modes

Card Mode![](/img/en-img/card.png)：Click to display all projects on the main interface in card layout.

List Mode![](/img/en-img/list.png)：Click to display all projects on the main interface in list layout.

Project Status Icons：![](/img/en-img/waiting.png)![](/img/en-img/running.png)![](/img/en-img/stoped.png)![](/img/en-img/error.png)

Waiting（Waiting for reconstruction）、Running（Reconstruction in progress）、Stoped（Reconstruction suspended）、Error（Reconstruction failed）。

Results：![](/img/en-img/3DGS.png)

3D（Contains 3D reconstruction outputs）、GS（Contains Gaussian Splatting outputs）。

### Multi-select Projects

![](/img/en-img/select2.png)Click the multi-select icon to enter multi-selection mode.

- Single Selection: Click project cards to select individual projects.
- Select All: Select all projects.
- Clear Selection: Deselect all currently chosen projects.
- Export Projects: Export selected projects to a specified folder in .mprj format.
- Delete Projects: Remove all selected projects.



![](/img/en-img/select.png)


### Import Project

![](/img/en-img/import.png)Click Import Project, select an .mprj project file, and import it to the current project list.



### New Project

![](/img/en-img/newproject.png)Click New Project, enter a project name, and select the initial task type based on your data type.

![](/img/en-img/newproject2.png)

### Project Options

![](/img/en-img/options.png)Click the icon at the bottom right of a project card to expand the project option menu.
- Copy: Copy the project parameters and aerial triangulation results.
- Export: Export the project to a specified folder in .mprj format.
- Rename: Modify the project name.
- Tag Management: Add or remove tags for the project.
- Delete: Permanently remove the project.

![](/img/en-img/options2.png)
