## Preparation Before Use

You need to activate the software and complete essential preference settings before using it.

### Software Activation

Visit the official website (https://na.mipmap3d.com/), log in to your account, navigate to the MipMap Lite product page, and click Apply for Trial.



Go to License Management in your Personal Center, locate the license for the corresponding version, and click Copy Activation Code.




Click the Settings icon at the top-right corner of the software to open the Settings panel.

![](/img/en-img/License.png)


Select License Management, paste the activation code, and click Bind to activate the software.


![](/img/en-img/License2.png)



### Working Directory Configuration

The default working directory of the software is C:\Users\Administrator\Documents\mipmap-lite. It stores intermediate files generated during project reconstruction and final model outputs, which will occupy a large amount of disk space.

We recommend that you reassign the working directory to a disk with ample free space after your first login. SSD storage delivers optimal performance.


![](/img/en-img/workingdirectory.png)


When changing the working directory, you may choose whether to migrate existing project files. If checked, all files from the current directory will be moved to the new path; this process may take a long time for large file sets.


![](/img/en-img/workingdirectory2.png)


### PC Environment Check

For optimal performance, set your power plan to Best Performance, and configure all screen and sleep power options to Never.

![](/img/en-img/power.png)
