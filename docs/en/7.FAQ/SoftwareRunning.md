## Software running

### Antivirus blocking installation

Cause: The installer is not code-signed. Software downloaded from official channels does not contain malware.

Solution: Add the installation directory to your antivirus whitelist. If files have been quarantined, restore them from quarantine.

### Connection failure — network/server issues

![](/img/en-img/networkerror.png)

Solution: Check your network connection and verify that VirBox is installed and running.

Check the VirBox application status and start the `virbox` service manually; if there are issues, use the one-click repair option.



### License Binding failed

![](/img/en-img/servererror.png)

Solution: Verify there are no leading/trailing spaces or extra characters in the activation code, and ensure you are using the same account for binding.
